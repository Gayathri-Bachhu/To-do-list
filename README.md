# ✅ To-Do List Web App

A simple and interactive To-Do List built using **HTML**, **CSS**, and **JavaScript**. Stay organized by adding, completing, and removing tasks with ease!

# 💡 Features
- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- 🗑️ Delete tasks  
- 💻 Fully responsive UI

# 🖥️ Tech Stack
- **HTML** for structure  
- **CSS** for styling  
- **JavaScript** for interactivity
